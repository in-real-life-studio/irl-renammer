Renommage
Recherche un texte dans chaque nom de fichier et le remplace.
Exemple : "IMG" → "Photo" renomme IMG_001.jpg en Photo_001.jpg
Renommage par expression reguliere avec groupes de capture.
Exemple : pattern (\w+)-(\d+) + remplacement $2_$1 transforme shot-042 en 042_shot
Ajoute du texte au debut et/ou a la fin de chaque nom de fichier (avant l'extension).
Exemple : prefixe FINAL_ renomme comp.exr en FINAL_comp.exr
Ajoute un numero sequentiel a chaque fichier. Configurez le debut, le pas, le nombre de chiffres (padding) et la position.
Exemple : debut 1, padding 3, prefixe → 001_fichier.jpg, 002_fichier.jpg, ...
Change la casse de chaque nom de fichier. Detecte automatiquement les mots (espaces, underscores, tirets, camelCase).
Exemple : snake_case transforme Mon Fichier en mon_fichier
Decoupe le nom par un separateur et garde uniquement les segments choisis. Ideal pour les fichiers generes (IA, exports) avec une structure PREFIX_PARAM1_PARAM2_..._timestamp.
Exemple : separateur _, garder 1-3, ajouter _v01 → TARGET_UBIGI_CDM26_Gemini3_..._12-55-34.png devient TARGET_UBIGI_CDM26_v01.png
Uniformise le padding des nombres dans les noms de fichiers. Ideal pour les sequences d'images dont la numerotation passe de 4 a 5 chiffres.
Exemple : render.0001.exr → render.00001.exr (auto-detecte le padding necessaire)
